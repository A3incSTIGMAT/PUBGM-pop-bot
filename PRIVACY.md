# Политика конфиденциальности NEXUS Chat Manager

**Дата последнего обновления:** 13 апреля 2026 г.

## 1. Какие данные собираются

- Telegram ID
- Имя пользователя (username)
- Имя и фамилия
- Баланс монет (NCoin)
- Статистика игр (победы/поражения)
- Данные анкеты (если заполнены)

## 2. Как используются данные

- Для работы игр и экономической системы
- Для сохранения прогресса пользователя
- Для упоминаний в общем сборе (/all)
- Для реферальной системы

## 3. Хранение данных

- Данные хранятся в зашифрованной базе данных SQLite
- Сервер расположен на платформе Amvera (Россия)
- Данные не передаются третьим лицам

## 4. Срок хранения

- Данные хранятся до тех пор, пока вы используете бота
- Вы можете запросить удаление данных командой /delete_my_data

## 5. Ваши права

- Запросить копию ваших данных
- Исправить неточные данные
- Удалить все ваши данные
