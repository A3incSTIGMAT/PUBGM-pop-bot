# NEXUS-bot

Мощный чат-менеджер для Telegram с играми, экономикой и модерацией.

## Возможности

### 🛡 Модерация
- `/all` — отметить всех участников
- `/ban` — забанить пользователя
- `/mute` — заглушить на время

### 💰 Экономика
- `/balance` — ваш баланс
- `/gift` — подарить монеты
- `/top` — топ пользователей

### 🎮 Игры
- `/rps` — камень-ножницы-бумага
- `/roulette` — рулетка
- `/duel` — дуэль (скоро)

## Установка

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать бота у @BotFather
4. Заполнить `.env` файл
5. Запустить: `python bot.py`

## Переменные окружения

- `BOT_TOKEN` — токен бота от @BotFather
- `ADMIN_IDS` — ID глобальных админов (через запятую)
